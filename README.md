# Java Chatbot (with Apache OpenNLP)

A Java console chatbot built with Maven, using Apache OpenNLP
for natural language processing.

## Status
🚧 Work in progress — actively adding features.

## Goal
Long-term goal is to turn this into a resume-analyzing AI tool
that uses NLP to extract and evaluate information from resumes.

## Tech Stack
Java, Maven, Apache OpenNLP

## How to Run
1. Clone the repository
2. Make sure Java and Maven are installed
3. Build: `mvn clean package`
4. Run: `mvn exec:java` (veya senin çalıştırdığın komut neyse onu yaz)
